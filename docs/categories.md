# Organizing with Categories

Categories allow you to see where your money is actually going. Instead of just seeing "Starbucks", you'll see "Food & Groceries".

## The `categories` Tab

Firesheet comes with a set of default categories, but you can change these at any time. Each category has:
- **Name**: e.g., "Transport", "Salary", "Housing".
- **Icon**: An emoji or symbol to make your charts look great! 🚌 💰 🏠

## Automatic Categorization

The real magic happens when you let Firesheet do the work for you. You can define **keywords** for each category. When you import a transaction, Firesheet looks at the description and contra-account. If it finds a match, it tags the category automatically.

### How to set up Keywords
1. Go to the `categories` tab.
2. Find the row for your category.
3. Add keywords separated by commas (e.g., for "Food & Groceries", you might add `Lidl, Tesco, Aldi, Waitrose`).

### Pro Tips for Keywords
- **Be Specific**: If you use a broad keyword like "Bank", it might match "Bank Transfer" as well as "Bank of America".
- **Case Doesn't Matter**: "Lidl" will match "LIDL" or "lidl".

## Why Bother?
Once your transactions are categorized, the **Dashboard** and **Charts** in your Firesheet will come to life, showing you exactly how much you spend on coffee versus rent each month!

---

::: tip Already categorized?
If you manually change a category in the `source` tab, Firesheet will remember your choice and won't overwrite it during the next automation run.
:::
